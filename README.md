# Minesweeper для Windows/macOS

## Описание

Данный проект влючает в себя две версии игры **Minesweeper** для macOS и Windows.

## Технологии

* Язык: (macOS: Swift; Windows: C#)
* UI: (macOS: SwiftUI; Windows: WinForm)
